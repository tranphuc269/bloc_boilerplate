Đây là kiến trúc dự án project sưu tầm từ https://github.com/wem2017/flutter_bloc_boilerplate, vào link để lại star để ủng hộ tác giả

Open with Editer VSCode or Android Studio and run

## Boilerplate Features:

* Splash
* Login
* Home
* Routing
* Theme
* Dio
* Database
* Validation
* Logging
* Multiple Theme and Dark Theme
* Multiple language
* Multiple font
* CI/CD github action

### Folder Structure
Here is the core folder structure which flutter provides.

```
flutter_bloc_boilerplate/
|- android
|- build
|- ios
|- lib
```

Here is the folder structure we have been using in this project

```
lib/
|- api/
|- blocs/
|- configs/
|- database/
|- models/
|- repositorys/
|- screens/
|- utils/
|- widgets/
```
